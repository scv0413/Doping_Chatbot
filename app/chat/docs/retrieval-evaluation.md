# Retrieval Evaluation

## 목적

이 문서는 `retrieval_inspector.py`의 검색 결과를 사람이 평가한 기록이다.

RAG 답변 품질을 보기 전에 먼저 retriever가 올바른 근거 chunk를 찾는지 확인한다.

평가 기준은 다음과 같다.

```text
충분: 현재 검색 결과만으로 답변 가능
부분적으로 충분: 관련 근거는 있으나 순위나 누락이 있어 보완 필요
부족: 사용자가 기대하는 핵심 근거가 검색되지 않음
```

## 평가 요약

| 질문 | 평가 | 핵심 판단 | 개선 방향 |
|---|---|---|---|
| S0 비승인약물이 뭐야? | 충분 | WADA 금지목록 S0 정의가 rank 1~2에 검색됨 | 낮음 |
| TUE는 누가 신청해야 해? | 질문 수정 | 실제 의도는 신청 주체보다 신청 방법에 가까움 | 질문을 “TUE는 어떻게 신청해야 해?”로 변경 |
| 도핑검사를 거부하면 어떤 불이익이 있어? | 부족 | 거부/회피/실패에 대한 직접 근거가 부족함 | query rewriting, 현장 대응 manual 필요 |
| 검사관 신분이 불분명하면 어떻게 해야 해? | 부족 | 신분 확인 방법과 지연 가능 여부 근거 부족 | DCO/시료채취요원/통지 관련 근거 보강 |
| 경기기간 중 금지약물은 어떻게 확인해? | 부족 | 규정 책임은 검색되나 확인 방법 안내가 부족함 | KADA 약물검색 source 추가 |
| 새벽에 혈액검사를 요청받으면 어떻게 해야 해? | 부분적으로 충분 | 관련 근거가 rank 3~5에 있으나 답변 구성에는 보강 필요 | query rewriting, 공식 source 보강 |

## 1. S0 비승인약물이 뭐야?

### 현재 검색 결과

```text
rank 1: wada_prohibited_list_2026_ko:p5:c1
rank 2: wada_prohibited_list_2026_ko:p5:c0
```

### 평가

충분.

### 이유

WADA 금지목록의 S0 비승인약물 정의와 한국어 설명이 상위에 검색되었다.

### 권장 답변 방향

S0은 비승인약물로 상시 금지이며, 정부 보건기구에서 사람 치료용으로 승인하지 않은 약리적 물질이라고 설명한다.

### 추가 안내

사용자가 특정 제품명이나 성분명을 이어서 물으면, 해당 성분이 S0 또는 다른 금지분류에 해당하는지 추가 확인하도록 유도한다.

## 2. TUE는 누가 신청해야 해?

### 현재 검색 결과

```text
rank 1: kada_anti_doping_rules_2021_ko:p28:c1
rank 3: kada_anti_doping_rules_2021_ko:p28:c0
```

### 평가

질문 수정 필요.

### 이유

실제 사용자 관심은 “누가 신청하느냐”보다 “어떻게 신청하느냐”에 더 가깝다.

### 수정 질문

```text
TUE는 어떻게 신청해야 해?
```

또는

```text
TUE 신청 방법과 대리 신청 가능 여부를 알려줘.
```

### 권장 답변 방향

- TUE 신청 방법을 중심으로 설명한다.
- 선수 수준에 따라 신청 기관이나 기준이 조금이라도 다르면 구분해서 말한다.
- 선수로부터 위임받은 선수지원요원이 대리 신청할 수 있다는 내용을 포함한다.
- KADA 누리집에 게시된 온라인, 팩스, 이메일 등 공식 신청 방법 확인을 안내한다.

## 3. 도핑검사를 거부하면 어떤 불이익이 있어?

### 현재 검색 결과

```text
rank 1: kada_anti_doping_rules_2021_ko:p100:c1
rank 5: kada_anti_doping_rules_2021_ko:p100:c0
```

### 평가

부족.

### 이유

현재 검색 결과는 부정행위, 공격적 행위, 도핑관리 방해에 가까운 chunk를 찾았지만 사용자가 기대하는 “시료채취 거부/회피/실패 시 불이익”을 직접 설명하는 근거가 상위에 충분히 나오지 않았다.

### 사용자 의도

- 검사를 거부하면 어떤 불이익이 있는지 알고 싶다.
- 불합리하거나 어쩔 수 없는 상황에서 불이익을 피하려면 어떻게 행동해야 하는지 알고 싶다.

### 답변에 포함해야 할 내용

- 정당한 사유 없는 시료채취 거부, 회피, 실패는 도핑방지규정 위반으로 이어질 수 있다.
- 욕설, 멱살, 검사관과 신체적 충돌, 현장 이탈, 무단 거부는 거부 또는 방해로 해석될 수 있다.
- 불합리한 상황이어도 감정적으로 대응하지 말고, 이의 제기 의사를 기록하고 동석자/팀 관계자/통역을 요청하는 방향으로 안내해야 한다.
- 검사관의 불합리한 행동에 대해 정당하게 문제 제기하거나 거부할 수 있는 조건과 절차도 별도 근거 확인이 필요하다.

### 개선 방향

```text
거부 → 시료채취 거부, 회피, 실패, Failure to Comply, Sample Collection
```

현장 대응 manual에 “거부로 오해받을 수 있는 행동”과 “정당한 이의 제기 방법”을 추가한다.

## 4. 검사관 신분이 불분명하면 어떻게 해야 해?

### 현재 검색 결과

```text
rank 1: kada_anti_doping_rules_2021_ko:p35:c1
rank 3: kada_anti_doping_rules_2021_ko:p40:c0
```

### 평가

부족.

### 이유

현재 검색 결과는 검사 권한 위임/위탁, 조사 권한 같은 넓은 규정 내용은 찾았지만 사용자가 기대하는 “검사관이 공식적으로 온 사람인지 확인하는 방법”을 직접적으로 제공하지 못했다.

### 사용자 의도

- 이 사람이 공식 검사관인지 확인하고 싶다.
- 신분이 불분명할 때 어떻게 안전하게 대응해야 하는지 알고 싶다.
- 검사관 신분을 완전히 인지할 때까지 검사를 지연시켜도 되는지 알고 싶다.

### 답변에 포함해야 할 내용

- 검사 자체를 바로 거부한다고 표현하지 않는다.
- 검사관의 신분증, 소속기관, 검사 권한, 통지 문서 확인을 정중히 요청한다.
- 팀 관계자, 트레이너, 통역, 보호자 등 동석을 요청할 수 있는지 안내한다.
- 확인 과정이 검사 거부 또는 검사 방해로 해석되지 않도록 표현과 행동에 주의해야 한다.
- 신분 확인이 완료될 때까지 검사를 지연시켜도 되는지에 대해서는 명확한 공식 근거 확인이 필요하다.

### 개선 방향

```text
검사관 신분 → 도핑검사관, 시료채취요원, DCO, Sample Collection Personnel, identification, notification
```

ISTI와 KADA 도핑검사 절차 문서에서 DCO identification, athlete notification 관련 조항을 보강한다.

## 5. 경기기간 중 금지약물은 어떻게 확인해?

### 현재 검색 결과

```text
rank 1: kada_anti_doping_rules_2021_ko:p19:c1
rank 2: kada_anti_doping_rules_2021_ko:p19:c0
```

### 평가

부족.

### 이유

현재 검색 결과는 경기기간 중 금지약물 검출 시 위반 가능성, 선수 책임, 시료채취 관련 조항을 찾았다. 하지만 사용자가 기대하는 것은 규정 설명이 아니라 “어디서 어떻게 확인하는지”에 대한 실용 안내다.

### 답변 방향

- 챗봇에 제품명 또는 성분명을 입력해 확인하도록 안내한다.
- KADA 홈페이지의 약물검색 메뉴에서 검색할 수 있다고 안내한다.
- 검색 결과가 불명확하거나 주사제/복용법/경기기간 여부/TUE 여부가 관련될 경우 추가 확인이 필요하다고 안내한다.

### 개선 방향

```text
금지약물 확인 → 약물검색, 제품명 검색, 성분명 검색, KADA 금지약물 검색
```

약물 확인형 질문은 규정 문서보다 KADA 약물검색 source를 우선 검색하도록 routing한다.

## 6. 새벽에 혈액검사를 요청받으면 어떻게 해야 해?

### 현재 검색 결과

```text
rank 3: kada_anti_doping_rules_2021_ko:p34:c0
rank 4: wada_isti_2021_ko_en:p135:c2
rank 5: kada_anti_doping_rules_2021_ko:p34:c1
```

### 평가

부분적으로 충분.

### 이유

새벽 검사와 관련된 핵심 근거가 rank 5에 검색되었다. 또한 시간과 장소를 불문한 시료 제공 요구 가능성은 rank 3에, 혈액 시료 관련 절차 일부는 rank 4에 검색되었다. 다만 중요한 근거가 top 1이 아니라 rank 3~5에 분산되어 있어 답변 구성에는 query rewriting 또는 reranking이 필요하다.

### 사용자 의도

가장 중요한 것은 “혈액검사 요청 시 안전하게 대응하는 법”이다.

### 현재 확인한 근거

- KADA 규정 chunk에는 검사권한이 있는 선수에게 시간과 장소를 불문하고 시료 제공을 요구할 수 있다는 내용이 있다.
- KADA 규정 chunk에는 오후 11시부터 오전 6시 사이 검사 제한, 특정 60분 time slot, 야간 검사 조건 관련 내용이 있다.
- WADA ISTI chunk에는 11 p.m.부터 6 a.m. 사이 overnight testing과 sample collection 거부 불가 관련 내용이 있다.
- KADA 도핑검사 절차 안내에는 선수에게 소변 또는 혈액 시료 채취를 통지한다고 설명한다.

### 답변에 포함해야 할 내용

- 혈액검사라는 이유만으로 즉시 거부한다고 안내하면 안 된다.
- 먼저 검사관의 신분, 소속, 권한, 통지 내용을 확인한다.
- 가능한 경우 트레이너, 팀 관계자, 통역 동석을 요청한다.
- 새벽 시간대라면 11 p.m.~6 a.m. 관련 규정과 60분 time slot 여부를 확인해야 한다.
- 불합리하다고 느껴도 현장 이탈, 욕설, 신체적 충돌, 무단 거부는 피해야 한다.
- 이의가 있으면 검사서 또는 기록에 남기고, 사후 KADA/소속 경기단체/국제연맹에 확인하도록 안내한다.

### urine 검사로 대체 가능한지에 대한 판단

현재 확인한 근거만으로는 “혈액검사를 소변검사로 대체할 수 있다”고 일반적으로 안내하면 안 된다.

KADA 도핑검사 절차에서는 소변 또는 혈액 시료 채취가 통지될 수 있다고 설명하고, WADA/KADA 규정은 시료 제공 의무와 혈액 시료 절차를 전제로 한다. 따라서 대체 가능 여부는 검사 목적, 검사권한, 시료 유형, 해당 검사 프로그램에 따라 달라질 수 있으므로 공식 근거 없이 단정하지 않는다.

### 개선 방향

```text
새벽 → 오후 11시, 오전 6시, 60분, 야간 검사, overnight testing
혈액검사 → 혈액 시료, blood sample, blood collection, BCO
거부 → refuse, failure to comply, sample collection
```

현장 대응 manual에 “새벽 혈액검사 요청” 상황을 별도 문서로 추가한다.

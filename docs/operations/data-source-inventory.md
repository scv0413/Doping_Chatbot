# Data Source Inventory

## 목적

이 문서는 챗봇이 사용할 데이터 소스를 정리한다.

각 자료는 RAG 문서로 사용할지, 구조화 데이터로 만들지, 별도 검색 tool로 연결할지 구분한다.

## 데이터 분류 기준

### 1. 문서 RAG 대상

규정, 절차, TUE, 제재, 선수 권리와 의무처럼 문맥과 조항 해석이 필요한 자료.

### 2. 구조화 데이터 대상

약물명, 성분명, 영어명, 제품명, 투여 경로, 종목별 금지 여부처럼 필드 단위 검색이 중요한 자료.

### 3. 별도 Tool 대상

KADA 금지약물 검색처럼 최신성, 검색 조건, 외부 서비스 구조가 중요한 자료.

### 4. 확인 필요 대상

공식 근거를 찾기 전까지 챗봇이 단정하면 안 되는 자료.


## 우선 수집 자료

| 우선순위 | 자료 | 출처 | 사용 방식 | 이유 | 상태 |
|---|---|---|---|---|---|
| 1 | KADA 도핑방지규정 | KADA | 문서 RAG | 검사 거부, 제재, 권리와 의무 근거 | 확인 필요 |
| 2 | KADA 도핑검사 절차 자료 | KADA | 문서 RAG | 검사관 신분, 검사 절차, 지연 가능성 확인 | 확인 필요 |
| 3 | KADA TUE 안내 | KADA | 문서 RAG | 치료목적사용면책 절차 안내 | 확인 필요 |
| 4 | WADA Prohibited List | WADA | 문서 RAG + 구조화 | 금지약물, 금지방법, 종목별 금지 확인 | 확인 필요 |
| 5 | KADA 금지약물 검색서비스 | KADA | 별도 Tool | 제품명/성분명 검색 | 확인 필요 |
| 6 | KADA 보충제 안내 | KADA | 문서 RAG | 보충제 오염, 미표시 성분 위험 안내 | 확인 필요 |
| 7 | KADA 한약 관련 자료 | KADA | 문서 RAG | 한약재 위험 안내 | 확인 필요 |
| 8 | WADA International Standard for Testing and Investigations | WADA | 문서 RAG | 검사 절차, 혈액/소변 검사, 선수 권리 확인 | 확인 필요 |
| 9 | 약학정보원 또는 공신력 있는 약물 DB | 외부 | 구조화 데이터 | 제품명-성분명-영어명 매핑 | 확인 필요 |

## 반드시 공식 문서로 확인해야 할 질문

- 혈액검사를 정당하게 거부할 수 있는 조건이 있는가?
- 혈액검사를 소변검사로 대체 요청할 수 있는가?
- 새벽 또는 예상하지 못한 시간의 혈액검사 절차는 어떻게 되는가?
- 응급 의료 상황에서 도핑검사를 지연하거나 조정할 수 있는가?
- 검사관 신분 확인이 불충분할 때 선수는 어떻게 행동해야 하는가?
- 동석자, 통역, 팀 닥터 요청은 어떤 범위에서 가능한가?
- 사후 TUE 또는 소급 TUE가 가능한 조건은 무엇인가?
